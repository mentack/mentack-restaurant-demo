# **App Name**: Mentack Restaurant

## Core Features:

- Multi-page Routing: Implement multi-page routing for Home, Menu, Menu Category, About, Gallery, Reservations, and Contact pages.
- Bilingual Support: Implement a language toggle for Turkish and English, persisting the selected language via localStorage and utilizing a translation dictionary.
- SEO Optimization: Ensure each page is SEO-ready with title, meta description, and OpenGraph tags.
- Image Optimization: Optimize images for fast loading with lazy loading for content below the fold.
- Reservation Form: Implement a reservation form with validation and a success message.
- Dynamic Menu Display: Generate and display Menu categories and dishes, populating them from a database.

## Style Guidelines:

- Primary color: Deep charcoal (#343A40) to convey elegance and sophistication.
- Background color: Warm, light neutral (#F8F9FA) off-white/cream to provide a welcoming atmosphere.
- Accent color: Muted gold (#A38745), used subtly for highlights and CTAs.
- Headings: 'Playfair', serif, elegant, fashionable, for headlines.
- Body: 'PT Sans', sans-serif, clean, readable, for body text.
- Use minimal, line-based icons for a clean and modern look.
- Employ a max-width container of approximately 1120px to ensure content is well-structured and visually appealing on larger screens.
